# Common

This directory contains scripts that are meant to be reused across multiple projects.

You can integrate them in your github project by forking the following repository:
- SSH : git@github.com:OmarAboudou/Godot_Utils.git
- HTTPS : https://github.com/OmarAboudou/Godot_Utils.git

## Warning

This directory is coming from a forked repository. 
Please be very cautious when modifying it 
as it might result in the forked repository 
being unable to synchronize with the base repository.
